1. Tu Primera Orden: `print()`

Para decirle a la computadora que muestre un mensaje en la pantalla, usamos la orden (o función) print(). El mensaje que quieres mostrar va entre paréntesis y comillas.

2. Guardando Información: Variables

Imagina que tienes cajas mágicas donde puedes guardar cosas y ponerles una etiqueta. En programación, esas "cajas" se llaman variables. Usamos variables para guardar datos que nuestro programa necesitará más tarde.

Para crear una variable, le das un nombre (la etiqueta de la caja) y usas el signo igual (=) para asignarle un valor (lo que guardas en la caja).

nombre_heroe = "Iron Man" # Texto (se llama string o cadena)
poder_heroe = 90          # Número entero (se llama integer o int)
altura_heroe = 1.85       # Número con decimales (se llama float)

print(nombre_heroe)
print(poder_heroe)
print(f"La altura de {nombre_heroe} es: {altura_heroe} metros.") # ¡Mira esta f-string!
Nombres de Variables:
  • Deben empezar con una letra o un guion bajo (_).
  • Pueden contener letras, números y guiones bajos.
  • Python distingue mayúsculas de minúsculas (edad no es lo mismo que Edad).
  • Usa nombres descriptivos, como puntos_del_juego en lugar de solo p.

Python es inteligente y sabe con qué tipo de dato está trabajando. Los más comunes que usaremos al principio son:

2. ¡A Calcular! Operaciones Matemáticas

Python puede actuar como una potente calculadora. Estos son los operadores matemáticos básicos:

  • + Suma
  • - Resta
  • * Multiplicación
  • / División (siempre da como resultado un float)
  • // División Entera (descarta la parte decimal)
  • % Módulo (resto de la división)
  • ** Exponenciación (elevar a una potencia)

a = 10
b = 3
a_b_Suma = a + b			# Suma
a_b_Multiplicacion = a * b		# Multiplicación
a_b_Division = a / b			# División
a_b_Div_Entera = a // b			# División entera
a_b_Modulo = a % b			# Módulo (muestra el resto de la división. Ej.: 10 dividido 3 es 3 con resto 1)
a_b_Exponenciacion = a ** b		# Exponenciación (10 elevado a 3)


# Mostramos los resultados
print(f"{a} + {b} = {a_b_Suma}")    		# Suma
print(f"{a} * {b} = {a_b_Multiplicacion}")	# Multiplicación
print(f"{a} / {b} = {a_b_Division}")   	 	# División
print(f"{a} // {b} = {a_b_Div_Entera}")  	# División Entera
print(f"{a} % {b} = {a_b_Modulo}")    		# Módulo
print(f"{a} ** {b} = {a_b_Exponenciacion}") 	# Exponenciación

Practica: Calcula el área de un rectángulo con base 5 y altura 8. Luego, calcula cuánto es 7 elevado al cubo.

Sección 2: Tomando Decisiones (if, else)

Clase 3: Tomando Decisiones con `if` (Condiciones)

¡Hola de nuevo! Hasta ahora, nuestros programas han seguido instrucciones una tras otra, en orden. Pero, ¿qué pasa si queremos que el programa tome decisiones y haga cosas diferentes según la situación? Para eso, Python nos da una herramienta muy poderosa: la declaración if (que significa "si" en inglés).

1. ¿Qué es una Condición?

Imagina que le dices a un amigo: "Si llueve, llevaré un paraguas". La parte "llueve" es una condición. Es algo que puede ser verdadero o falso.

En Python, las condiciones también son expresiones que evalúan a Verdadero (True) o Falso (False). Para crear estas condiciones, usamos operadores de comparación:

  • == : ¿Es igual a? (Ej: edad == 18)
  • != : ¿Es diferente de? (Ej: nombre != "Ana")
  • < : ¿Es menor que? (Ej: temperatura < 0)
  • > : ¿Es mayor que? (Ej: puntos > 100)
  • <= : ¿Es menor o igual que? (Ej: intentos <= 3)
  • >= : ¿Es mayor o igual que? (Ej: nivel >= 5)
¡Ojo con la Igualdad! No confundas == (para comparar si dos cosas son iguales) con = (para asignar un valor a una variable). ¡Es un error muy común al empezar!

2. La Estructura de un `if`

Una declaración if básica en Python se ve así:


if condicion_a_evaluar:
    # Bloque de código que se ejecuta SI la condición es Verdadera (True)
    # ¡Importante! Este bloque DEBE estar indentado (tener un espacio a la izquierda).
    print("La condición es verdadera.")

# Este código se ejecuta siempre, porque está fuera del bloque if (sin indentación)
print("Esta línea se ejecuta después del if, sin importar la condición.")

Desglose:

  1. Palabra clave if.
  2. Luego, la condicion_a_evaluar (que usa operadores de comparación).
  3. Termina la línea de condición con dos puntos :.
  4. En la siguiente línea (o líneas), el código que quieres que se ejecute solo si la condición es True. Este código debe estar indentado (generalmente 4 espacios o una tabulación).

¡La Indentación es Clave!

Python usa la indentación (los espacios al inicio de una línea) para saber qué código pertenece a qué bloque (como el de un if). Si no indentas correctamente, ¡tu programa dará error o no hará lo que esperas!

Sección 2: Tomando Decisiones (if, else)

Clase 4: Más Decisiones con `else` y `elif`

En la clase anterior aprendiste a usar if para que tu programa ejecute código solo cuando una condición es verdadera. Pero, ¿qué pasa si quieres hacer algo cuando la condición es falsa? ¿O si tienes varias condiciones que probar?

¡Para eso tenemos else ("si no") y elif ("si no, si...")!

1. Cuando la Condición `if` no se Cumple: `else`

Imagina la situación: "Si llueve, llevaré paraguas. Si no (else), llevaré lentes de sol."

La cláusula else se coloca después de un bloque if. El código dentro del bloque else se ejecuta únicamente si la condición del if es Falsa (False).

Estructura:


if condicion:
    # Código si la condición es True
    print("La condición del if es Verdadera.")
else:
    # Código si la condición del if es False
    print("La condición del if es Falsa, así que se ejecuta el else.")
¡Importante! Un else siempre va acompañado de un if (o un elif, como veremos). No puede existir un else solito. Y, al igual que con if, el bloque de código de else debe estar indentado.

Ejemplo: ¿Puedes votar?

2. Múltiples Caminos: `elif` (Si no, si...)

A veces tenemos más de dos posibilidades. Por ejemplo: "Si la nota es 9 o más, es Sobresaliente. Si no, si (elif) la nota es 7 o más, es Notable. Si no, si (elif) la nota es 5 o más, es Aprobado. Si no (else), es Suspenso."

elif es una abreviatura de "else if". Te permite comprobar otra condición si la condición del if (o elif anteriores) fue Falsa.

Estructura:


if condicion1:
    # Código si condicion1 es True
    print("Se cumplió la condición 1.")
elif condicion2:
    # Código si condicion1 es False Y condicion2 es True
    print("No se cumplió la 1, pero sí la condición 2.")
elif condicion3:
    # Código si condicion1 y condicion2 son False Y condicion3 es True
    print("No se cumplieron ni la 1 ni la 2, pero sí la 3.")
else:
    # Código si NINGUNA de las condiciones anteriores (if/elif) fue True
    print("No se cumplió ninguna condición.")
Orden y Exclusión: Python evalúa las condiciones if y elif en orden. Tan pronto como encuentra una condición Verdadera, ejecuta su bloque de código y OMITE el resto de los elif y el else de esa cadena. ¡Solo se ejecuta UN bloque de toda la estructura `if-elif-else`!

Clase 5: Repitiendo Tareas con Bucles `for` e Introducción a las Listas

¡Bienvenido/a de nuevo! Imagina que necesitas escribir "Hola" en la pantalla 100 veces. ¿Lo escribirías 100 veces con print()? ¡Sería muy tedioso! Los programadores somos eficientes, así que usamos bucles para repetir tareas automáticamente.

Hoy aprenderás sobre el bucle for y un nuevo tipo de dato muy útil: las listas.

1. ¿Por Qué Necesitamos Bucles?

Los bucles nos permiten ejecutar un bloque de código múltiples veces sin tener que escribirlo repetidamente. Esto hace nuestros programas más cortos, más fáciles de leer y menos propensos a errores.

2. El Bucle `for` con `range()`: Repetir un Número Fijo de Veces

Una forma común de usar el bucle for es con la función range(). Esta función genera una secuencia de números.

Estructura básica:


# La variable 'i' tomará cada valor generado por range() en cada repetición (o iteración)
for i in range(numero_de_veces):
    # Código a repetir
    print(f"Esta es la repetición número: {i}")
  • range(N): Genera números desde 0 hasta N-1. Por ejemplo, range(3) genera 0, 1, 2.
  • range(inicio, fin): Genera números desde inicio hasta fin-1. Ej: range(2, 5) genera 2, 3, 4.
  • range(inicio, fin, paso): Genera números desde inicio hasta fin-1, incrementando en paso. Ej: range(1, 10, 2) genera 1, 3, 5, 7, 9.

Clase 6: Bucles `while` (Mientras) y Más Sobre Listas

Ya conoces el bucle for, ideal para repetir tareas un número conocido de veces o para recorrer elementos de una lista. Pero, ¿qué pasa si quieres repetir algo MIENTRAS una condición sea verdadera, sin saber exactamente cuántas veces será?

¡Para eso está el bucle while! Además, hoy aprenderemos más formas de trabajar y modificar nuestras listas.

1. El Bucle `while`: Repetir Mientras Algo Sea Cierto

Un bucle while (mientras) ejecuta un bloque de código repetidamente siempre y cuando una condición especificada se evalúe como Verdadera (True).

Estructura:


while condicion_a_evaluar:
    # Bloque de código que se ejecuta MIENTRAS la condición sea True
    # ¡Importante! Algo dentro de este bloque debe, eventualmente,
    # hacer que la 'condicion_a_evaluar' se vuelva False para evitar un bucle infinito.
    print("La condición sigue siendo verdadera...")
    # ... actualizar variables que afectan la condición ...

print("La condición se volvió falsa, el bucle while terminó.")
¡Cuidado con los Bucles Infinitos! Si la condición de un bucle while NUNCA se vuelve falsa, el bucle se ejecutará para siempre (o hasta que detengas el programa manualmente). Esto se llama un bucle infinito. Asegúrate siempre de que algo dentro del bucle cambie el estado de la condición.

2. `while` para Validar Entradas del Usuario

Un uso muy común de `while` es seguir pidiendo una entrada al usuario hasta que ingrese algo válido.

3. Más Formas de Trabajar con Listas

Las listas son muy flexibles. Aquí algunas operaciones comunes:

  • mi_lista.append(elemento): Añade elemento al FINAL de la lista.
  • len(mi_lista): Devuelve el número de elementos en la lista (su longitud).
  • mi_lista[indice] = nuevo_valor: Modifica el elemento en la posición indice.
  • mi_lista.pop(): Elimina y devuelve el ÚLTIMO elemento de la lista.
  • mi_lista.pop(indice): Elimina y devuelve el elemento en la posición indice.
  • elemento in mi_lista: Verifica si elemento está presente en la lista (devuelve True o False).

Clase 7: Organizando Datos con Diccionarios

¡Hola! Hasta ahora hemos usado variables para guardar datos individuales y listas para guardar colecciones ordenadas de datos. Hoy conoceremos otra forma muy poderosa de organizar información: los diccionarios.

1. ¿Qué es un Diccionario?

Imagina un diccionario de idiomas real: buscas una palabra (clave) y encuentras su significado (valor). Un diccionario en Python funciona de manera muy similar.

Un diccionario es una colección de elementos que NO están ordenados por posición (como las listas), sino que cada elemento tiene una clave única asociada a un valor. Se usan para almacenar datos que tienen una relación natural de "etiqueta" y "dato".

Se definen usando llaves {} y los pares clave-valor se separan por dos puntos :, mientras que los pares se separan por comas ,.

Sintaxis: mi_diccionario = {"clave1": valor1, "clave2": valor2, "clave_texto": "un string"}

  • Las claves suelen ser strings (textos) o números. Deben ser únicas dentro de un diccionario.
  • Los valores pueden ser de cualquier tipo: strings, números, booleanos, ¡e incluso otras listas o diccionarios!

Ejemplo Básico: Información de un Estudiante


# Creando un diccionario
estudiante = {
    "nombre": "Ada Lovelace",
    "edad": 28,
    "curso": "Introducción a la Computación",
    "es_programadora": True
}

print(estudiante)

2. Acceder, Añadir y Modificar Elementos

  • Acceder a un valor: Usas la clave entre corchetes: diccionario["clave"].
  • Añadir un nuevo par clave-valor: Simplemente asignas un valor a una nueva clave: diccionario["nueva_clave"] = nuevo_valor.
  • Modificar un valor existente: Asignas un nuevo valor a una clave existente: diccionario["clave_existente"] = valor_actualizado.

Ejemplo Práctico:

3. Recorriendo (Iterando) un Diccionario

Puedes usar un bucle for para recorrer los elementos de un diccionario de varias maneras:

  • Iterar sobre las claves (por defecto): 
    for clave in mi_diccionario:
    print(f"Clave: {clave}, Valor: {mi_diccionario[clave]}")
  • Iterar sobre los valores con .values(): 
    for valor in mi_diccionario.values():
    print(f"Valor: {valor}")
  • Iterar sobre los pares clave-valor con .items() (¡muy útil!): 
    for clave, valor in mi_diccionario.items():
    print(f"Clave: {clave} ---> Valor: {valor}")

      • Si es menor: Dar la pista "Demasiado bajo".
      • Si es mayor: Dar la pista "Demasiado alto".
  6. (Opcional) Si el bucle termina porque se acabaron los intentos, informar al jugador que perdió y cuál era el número.

Conceptos de Python a Utilizar: `import`, `random.randint()`, `print(), input(), conversión de tipos (int()), variables (incluyendo booleanas), bucle `while`, `if/elif/else`, operadores de comparación.

Pistas:
